# Discover Motherland Africa

A premium, interactive, and responsive web application built with Next.js, TypeScript, and TailwindCSS (configured dynamically).

## Features

- **Dynamic Hero Section**: Vibrant visuals and engaging call-to-actions.
- **Interactive Galaxy Footer**: Interactive particle/galaxy effect using modern canvas-based components.
- **Responsive Layout**: Seamless navigation across all desktop, tablet, and mobile devices.
- **Enquiry Form**: Interactive validation and seamless communication pathways.
- **Social Integration**: High-fidelity brand SVG icons for WhatsApp, Calls, Facebook, Instagram, and TikTok.

## Tech Stack

- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS & Custom CSS
- **Interactions**: Framer Motion & Custom Canvas Animations

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Automated deployments are set up via GitHub Actions to deploy static builds to GitHub Pages.
